# Walmart Predictive Analysis

## Project Overview
This project predicts Walmart weekly sales using historical sales data and Linear Regression. It demonstrates the complete Data Analytics workflow from data preprocessing to machine learning and dashboard visualization.

## Tools & Technologies
- Microsoft Excel
- Power Query
- MySQL
- Python (Google Colab)
- Scikit-learn
- Power BI

## Project Workflow
- Data Cleaning & ETL
- SQL Analysis
- Feature Engineering
- Linear Regression Model
- Model Evaluation
- Interactive Power BI Dashboard

## Machine Learning
- Algorithm: Linear Regression
- Train-Test Split: 80:20

## Evaluation Metrics
- MAE
- MSE
- RMSE
- R² Score

- FinalDataset.csv
